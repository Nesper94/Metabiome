# PhagePipe
Pipeline for metagenomics of rhizosphere and associated phages.

## Pre-processing of reads
- FastQC
- Trimmomatic
- Bowtie2

## Functional profiling

## Genome assembly
- MetaSPADES
- MegaHit

### Evaluation of assembly
- MetaQUAST
